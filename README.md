## 🧠 About

This is the **first version of Kogemi**, a basic anime streaming test site.

It was built as an early prototype to test:

- Anime search system  
- Streaming integration  
- Episode navigation  

> ⚠️ UI is minimal and not polished

---

## ✨ Features

🔍 Anime search (AniList API)  
🎬 Streaming via external sources  
📺 Episode selection system  
⏭ Next / Previous episode controls  
🔁 Primary & Backup streaming servers  
⚡ Fast and lightweight  

---

## 🖥️ How It Works

1. Search for an anime  
2. Select a result  
3. System fetches:
   - Metadata (episodes)
   - IMDB ID  
4. Loads streaming player  
5. Choose episode & server  

---

## ⚙️ Tech Stack

- JavaScript

## ⚠️ Limitations

- ❌ No modern UI  
- ❌ Hardcoded episode fallbacks  
- ❌ Limited metadata accuracy  
- ❌ No authentication system
